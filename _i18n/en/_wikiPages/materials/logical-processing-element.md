<a name="Crafting"></a>

### Crafting

It is crafted using the [HT Crafting Table]({{ "/wiki/ht-crafting-table" | relative_url }}):  

- Redstone x 20  
- Gold Ignot x 8  
- Nether Quartz x 4  
- [Reinforced ceramic plate]({{ "/wiki/ceramic-plate" | relative_url }}) x 1  
- [Star Dust]({{ "/wiki/star-dust" | relative_url }}) x 45
