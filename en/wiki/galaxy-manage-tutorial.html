How to create a Galaxy

On the Space Ship Click Space Ship Controller on the table
Click Find unknown galaxy

Type Galaxy Name in Chat

Click Confirm

How to open Galaxy manage GUI

On the Space Ship Click Space Ship Controller on the table
Switch to Joined galaxy using the button on the top-right corner

Click Galaxy that you want to manage. You must be Owner or Admin of the Galaxy.

And then click Manage Galaxy

How to create Planet

Open Manage GUI
Click Find unknown planet

Click Planet Type

You can only can create 2 planets for now, second one is hot planet(Costs 1000 star dust). Type Planet Name in Chat ( Only accepts alphabet and number - _)

Click Confirm

If you see red text, it means you use not allowed string or it has been used.

How to kick member

Open Manage GUI
Click Manage member

Choose Member you want to kick

Click Remove member

Click Yes

How to add member

Open Manage GUI
Click Add member

Type Player Username in Chat

How to review request

Open Manage GUI
Click Join application

Click Player you want to review

Click Yes 或 No

How to change member permission

Open Manage GUI
Click Manage member

Select the Player you want to change

Click Change role

Click the role you want to give ( ADMIN / MEMBER )

How to change Galaxy name

Open Manage GUI
Click Rename

Type Other Galaxy Name in Chat

How to change Galaxy info

Open Manage GUI
Click Change info

Type Galaxy Info in Chat

How to change Galaxy notice

Open Manage GUI
Click Change Notification

Type Galaxy notice in Chat

you can use \n to make a new line
